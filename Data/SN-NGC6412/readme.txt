This folder contains the data of PSN J17292918+7542390 (here after SN-NGC6412) II-P low luminous core collapse supernova 
used for Jager et al., 2020, MNRAS.

Files:
PSN-NGC6412_Phot_BVRI_has.eps
BVRI light curves

PSN-NGC6412_bol_HAS.eps
bolometric light curves

SN-NGC6412data.txt
BVRI table (Latex format)

reference for the comperators: please see Jager et al., 2020
